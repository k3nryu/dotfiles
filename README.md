# Dotfiles for Linux
## Dependencies
- Zsh
- Git
- Util-linux-user
- Oh-my-zsh(zsh manager)
- Powerlevel10k(zsh theme)
- zsh-syntax-highlighting
- zsh-autosuggestions
- tmux
- alacritty

## Installation
1. Clone repositories to local.
```bash
git clone --recursive -b Linux https://github.com/k3nryu/dotfiles.git
```
2. Execute 'install.sh' shell script.
```bash
docfiles/install.sh
```
